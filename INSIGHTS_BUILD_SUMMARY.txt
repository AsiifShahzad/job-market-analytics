═══════════════════════════════════════════════════════════════════════════════
🎯 ARCHITECTURE DECISIONS
═══════════════════════════════════════════════════════════════════════════════

1. SQL-FIRST AGGREGATION
   ✅ All calculations done in PostgreSQL (GROUP BY, AVG, COUNT)
   ✅ Only aggregated results sent to frontend
   ✅ Memory efficient even with 1M+ jobs

2. MODULAR BACKEND
   ✅ Each insight type has dedicated function
   ✅ Easy to test, modify, extend
   ✅ Reusable query builders

3. REACT QUERY CACHING
   ✅ 5-minute stale time (real-time + cache performance)
   ✅ 10-minute cache retention
   ✅ Automatic background refetch

4. ACTIONABLE FORMAT
   ✅ Human-readable insights (not raw data)
   ✅ Labels for quick scanning (🔥 📈 💰)
   ✅ Percentage changes (not absolute numbers)
