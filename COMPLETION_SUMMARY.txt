THE 9-STEP ANALYTICS PIPELINE:

  ✅ STEP 1:  Data Cleaning & Normalization
     • Removes invalid jobs (missing descriptions)
     • Normalizes city names ("NYC" → "New York")
     • Validates salary logic (min ≤ max)
     • Location: src/analytics/rigorous_engine.py::clean_jobs_data()

  ✅ STEP 2:  Hybrid Skill Extraction (Exact + Fuzzy)
     • Exact word-boundary matching
     • Fuzzy matching for misspellings (85% threshold)
     • Noise filtering (≥3 appearances minimum)
     • Location: src/analytics/rigorous_engine.py::validate_and_extract_skills()

  ✅ STEP 3:  Skill Demand Analysis (Normalized by Category)
     • Organized by category (Language, DevOps, Frontend, etc.)
     • Normalized demand scores (0-1 scale)
     • Top 10 per category
     • Location: src/analytics/rigorous_engine.py::analyze_skill_demand()

  ✅ STEP 4:  Trending Skills Detection (Corrected WOW)
     • Week-over-week growth calculation
     • Statistical validation (both weeks ≥10 jobs)
     • Filters false 100%+ growth
     • Location: src/analytics/rigorous_engine.py::detect_trending_skills()

  ✅ STEP 5:  Salary Analysis (Bias Removal)
     • IQR-method outlier removal
     • Minimum: 30 salary samples per skill
     • Seniority-controlled analysis
     • Location: src/analytics/rigorous_engine.py::analyze_salary_insights()

  ✅ STEP 6:  Location Insights (Cleaned Geography)
     • Top 10 cities and countries
     • Remote job percentage
     • Location normalization
     • Location: src/analytics/rigorous_engine.py::analyze_market_locations()

  ✅ STEP 7:  Skill Combinations (High Sample Only)
     • Identifies valuable skill pairs
     • Minimum: 5 co-occurrences
     • Returns top 10
     • Location: src/analytics/rigorous_engine.py::analyze_skill_combinations()

  ✅ STEP 8:  Actionable Insights (High Confidence Only)
     • HIGH: ≥50 samples, MEDIUM: ≥20 samples, LOW: <20 samples
     • Includes reasoning for each insight
     • Sample size transparency
     • Location: src/analytics/rigorous_engine.py::generate_actionable_insights()

  ✅ STEP 9:  Data Quality Report (Complete Audit Trail)
     • Jobs before/after cleaning
     • Data retention percentage
     • Skills validated vs removed
     • Location: src/analytics/rigorous_engine.py::AnalyticsOutput
